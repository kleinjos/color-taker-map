Color-Taker Map:
Scroll down!


  • Every map on this page is shown as a styled iframe of this map layer.
    This is an early attempt to produce maps that allow the end user to participate in the visualization process. Check out all of our maps at MapBox.
  • How does it work?

  • This iframe contains the first part of the maps above, the color-blocker MapTile layer.
    The color blocker layer limits the extents of the user-defined background color to the boundaries of the USA. As you can see, the iframe background color fills the negative space.

  • This iframe contains the second part, the color-taker MapTile layer.
    The color-taker layer accepts some of the user-selected background color, according to the opacity gradient set by the map maker during the rendering process.
